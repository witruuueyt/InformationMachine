Simulink

The Simulink Interface is only available in Game4Automation Professional

The Simulink interface allows the direct connection between Simulink from MathWorks (https://de.mathworks.com/products/simulink.html) and Game4Automation. It creates a direct memory communication between the two tools for fast communication.

This video gives you a short overview:
https://youtu.be/jS-sRI6IvPI

Please note, that this Interface is currently in Beta. For Simulink you need the Game4Automation Interface block. It is for free available from MathWorks. You will need to send us an email to info@gam4automation.com and we will forward your request to MathWorks.

Interface configuration

To use the interface you need first to implement a Game4Automation Interface block into your Simulink modell. In this interface you must define all Inputs and Outputs that should be exchanged with Game4Automation.

To add the Simulink interface into your Game4Automation scene you need to select gam4automation > Add Object > Interface > Simulink (Pro)

After this Simulink can be startet and the Signals can be imported by pushing on the Button Import Signals. You will now see all Simulink Signals as Singal Objects underneath your Simulink interface and you can connect them to your Game4Automation objects.
